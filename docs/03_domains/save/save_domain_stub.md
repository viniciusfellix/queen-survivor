# Domínio — Save

## Estado atual

Save básico existe, mas ainda não está integrado ao resultado da run.

## Regra oficial futura

O save deve registrar:

- XP total.
- Dinheiro permanente.
- Queens desbloqueadas.
- Mapas concluídos.
- Configurações.
- SFW.
- Upgrades comprados com nível e custo.
- Última run/resumo.
- Recordes básicos.

## Importante

Reset de progressão será gratuito e sem penalidade.

Por isso, o save não deve guardar apenas atributos finais. Ele deve guardar compras/níveis/custos.
