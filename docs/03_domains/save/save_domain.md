# Domínio — Save

`SaveManager` salva em `user://queen_survivors_save.json`. O save básico registra progresso permanente, mapas concluídos, última run e records básicos previstos.

## Limitação conhecida

O JSON pode ser editado externamente. Proteção/criptografia/assinatura permanece pendente e deverá considerar migração/compatibilidade de saves.

O painel técnico permite resetar progresso para testes sem encerrar a run atual.
