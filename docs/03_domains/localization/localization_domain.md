# Domínio — Localização

`LocalizationManager` carrega atualmente `res://data/localization/pt_br.json` e retorna textos por chave. Novos textos de gameplay/UI devem possuir localization.

Idiomas previstos: português, inglês, espanhol, coreano, chinês e japonês. A possível migração para o sistema nativo Godot ainda não foi implementada.
