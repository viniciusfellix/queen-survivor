# Checklist do Game Designer

## Ao editar arma Gaia

- [ ] Shape alinhada ao visual.
- [ ] Acerta centro e bordas, não acerta fora.
- [ ] Dano físico/mágico esperado.
- [ ] Upgrade de área escala a shape.
- [ ] Upgrades de dano/cooldown/lifetime funcionam.

## Ao editar Goblin

- [ ] Hurtbox acompanha corpo.
- [ ] Ataque só ocorre ao tocar PlayerHurtbox.
- [ ] Delay/intervalo respeitados.
- [ ] Fraquezas correspondem ao planejado.
- [ ] Flash claro aparece.
- [ ] Morto não ataca nem recebe dano.

## Ao editar resultado/mapa

- [ ] Timer correto.
- [ ] Vitória multiplica/bonifica corretamente.
- [ ] Derrota não multiplica.
- [ ] Save registra resultado.

## Final

- [ ] Console sem erros/warnings novos.
- [ ] Logs detalhados desligados após validação.
