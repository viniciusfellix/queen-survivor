# Checklist de Auditoria Estrutural

## Scripts

- [ ] Sem funções/nodes/signals residuais.
- [ ] Sem nomes removidos em scripts/resources ativos.
- [ ] Sem logs legacy restantes fora de necessidade registrada.
- [ ] Funções relevantes comentadas.
- [ ] Duplicidades reais extraídas para base/helper.

## Cenas e resources

- [ ] Scripts e UIDs válidos.
- [ ] BodyCollision, Hitbox e Hurtbox separados.
- [ ] Definitions não duplicadas em cenas genéricas.
- [ ] IDs e localization válidos.
- [ ] Shapes alinhadas visualmente.

## Execução

- [ ] Boot sem warning/erro.
- [ ] Regressão aprovada.
- [ ] Logs detalhados desligados.
- [ ] Documentação atualizada.
